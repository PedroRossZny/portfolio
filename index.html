Painel de Chamadas

Painel de Chamadas LaborWaze é um sistema full-stack desenvolvido para otimizar o fluxo de atendimento em hospitais e clínicas, organizando chamadas de pacientes com histórico completo, controle inteligente de múltiplas chamadas e exibição transparente do profissional responsável.

Saiba mais...

O Sistema de Chamadas LaborWaze é uma solução full-stack desenvolvida para otimizar, padronizar e tornar mais eficiente o processo de chamadas de pacientes em hospitais, clínicas e unidades de saúde.

O projeto foi criado para resolver um problema real enfrentado diariamente no setor da saúde: a desorganização no fluxo de atendimento, a ausência de histórico estruturado e a falta de transparência nas chamadas.

A solução funciona em dois modos distintos:

Modo Independente:

Sistema próprio com interface dedicada, geração de relatórios em PDF, controle inteligente de chamadas e personalização visual conforme a identidade da unidade.

Extensão para e-SUS:

Integração via navegador que permite realizar chamadas diretamente do ambiente oficial, sem modificar o banco de dados do sistema governamental, funcionando como uma camada complementar segura.

Entre os principais diferenciais estão:

  • Controle de múltiplas chamadas (1ª, 2ª e 3ª tentativa com diferenciação visual)
  • Sistema anti-spam para evitar múltiplos cliques
  • Registro completo com horário, sala e profissional responsável
  • Geração de relatórios de fluxo
  • Exibição em painel digital com alerta sonoro
  • Estrutura escalável e preparada para expansão

O projeto foi desenvolvido com foco em desempenho, segurança e escalabilidade, utilizando Next.js, NestJS e PostgreSQL. O tempo médio de envio das informações ao painel é de aproximadamente 0,3 segundos em conexões convencionais.

Mais do que um sistema de chamadas, trata-se de uma ferramenta estratégica para organização, transparência e eficiência no atendimento em unidades de saúde.

Este projeto foi desenvolvido como trabalho final de um curso de HTML5 e CSS3, com o objetivo de consolidar os fundamentos do desenvolvimento front-end utilizando apenas tecnologias base da web: HTML, CSS e JavaScript puro.

  • Organização modular de estilos e scripts

Além de reforçar conceitos essenciais da web, o projeto marcou um passo importante na evolução como desenvolvedor, representando a transição do aprendizado teórico para a construção de interfaces completas e interativas.

JavaScript Estudos

Formação prática e estruturada em JavaScript moderno, organizada em fases progressivas que vão dos fundamentos da linguagem à aplicação de conceitos avançados como POO, modularização e programação assíncrona, visando consolidar domínio técnico e base sólida para desenvolvimento full-stack.

Este projeto representa minha jornada estruturada de aprofundamento em JavaScript, organizada em duas etapas complementares (separadas no GitHub, mas parte de uma mesma evolução técnica).

Fase 1 - Fundamentos da Linguagem

Consolidação das bases do JavaScript:

  • var, let, const
  • Tipos primitivos e estruturas condicionais
  • Estruturas de repetição
  • Funções tradicionais e arrow functions
  • Escopo e hoisting
  • Métodos fundamentais de arrays
  • Manipulação de strings
  • Exercícios práticos de lógica

O objetivo foi construir domínio real da linguagem antes de avançar para conceitos estruturais.

Fase 2 - Conceitos Avançados e Aplicação Prática

Evolução para um nível mais próximo do mercado:

  • Programação Orientada a Objetos (classes, herança, encapsulamento, polimorfismo)
  • Prototypes e funções construtoras
  • Modularização com ES Modules (import/export) e CommonJS
  • Programação assíncrona (callbacks, Promises, async/await)
  • Manipulação de DOM
  • Estruturação de pequenos sistemas simulando aplicações reais

Inclui simulações práticas como:

  • Sistema simples de loja / carrinho de compras
  • Manipulação dinâmica de interface
  • Simulações de requisições assíncronas

Objetivo do Projeto

Consolidar uma base sólida e estruturada em JavaScript moderno, desenvolvendo não apenas sintaxe, mas compreensão profunda de funcionamento, organização de código e padrões utilizados no desenvolvimento frontend e backend.

Esse projeto é a base técnica que sustenta meus projetos mais avançados em React, Next.js e Node.js.
